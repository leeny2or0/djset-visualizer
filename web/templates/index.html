How to export your history file from rekordbox

  1. 1
    Open Histories

    In the rekordbox left sidebar, open Histories and select the session for the date you want.

  2. 2
    Show the needed columns

    Right-click the track list header and enable the Track Title · Artist · BPM · Key columns.

  3. 3
    Export to a file

    Right-click the list → Export a playlist to a file, and save it as txt.

  4. 4
    Upload

    Drop the saved .txt into the upload box above. Encodings (UTF-16, etc.) are handled automatically.

Your file is only used for analysis and briefly stored in a temp folder.
