트랙리스트

아티스트 - 트랙명 형식이에요. 복사해서 사운드클라우드 업로드 소개글에 붙여넣으세요.

rekordbox에서 히스토리 파일 내보내는 방법

  1. 1
    히스토리 열기

    rekordbox 좌측 사이드바의 Histories(히스토리)를 펼쳐 원하는 날짜의 세션을 선택합니다.

  2. 2
    필요한 컬럼 켜기

    트랙 목록 헤더를 우클릭해 Track Title · Artist · BPM · Key 컬럼이 보이도록 체크합니다.

  3. 3
    파일로 내보내기

    목록 위에서 우클릭 → Export a playlist to a file(파일로 내보내기)를 선택하고, 형식을 txt(텍스트)로 저장합니다.

  4. 4
    업로드

    저장한 .txt 파일을 위 업로드 창에 올리면 끝입니다. 인코딩(UTF-16 등)은 자동으로 처리돼요.

파일은 이 페이지(브라우저) 안에서만 처리되며 어디에도 업로드되지 않습니다.
